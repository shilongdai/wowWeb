Introduction

Welcome to the WoW 3.3.5a private server. This server is powered by TrinityCore MMORPG engine. You are registered in the database. To play the game, download the WoW modified client from: Here

To run the game at optimal configuration, set the screen resolution according to your computer, and turn the graphics settings to maximum. Because WoW 3.3.5a is pretty old, it will not run at full screen successfully on modern hardware. However, you could achieve the same effect using bordered window mode without borders.
